## Famions

Famions is a character brand centered on a core cast of Filipino-inspired familiars.

Resident Famions artist: RageBoi.

Learn more at [famions.com](https://www.famions.com/).

## Codex Pet Package

This repository adapts the Famions core four characters into Codex custom pet packages:

- Adarna
- Diwata
- Duwende
- Sundo

Codex pet packaging and implementation: Joe Josue / AhensyaHQ.
